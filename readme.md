### IO格式

```
输入：
!!! 第一行输入空白区域数量n 模块数量m 限制条数l !!!
接下来n行，每行e个组(a,b)，表示空白区域在(a,b)这条向量的左侧
接下来m行，每行一个字符串表示模块代号，然后输入一个字符串s或是坐标，表示模块的连接点（另一个模块或固定座标）
接下来l行，每输入两个字符串，字母开头代表模块，数字开头代表边
输出：
前m行每行输出该模块摆放位置的对角端点
最后一行输出连线的总长度
```

### Dependencies
```
libcairo-dev
```

### Compile & run
```
make -j4
./main < ?.in
```
